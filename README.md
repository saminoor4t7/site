# 🏥 Medical Store System

> A smart pharmacy management app connecting patients, medical stores, and suppliers.

## 📌 Modules

| Module | Description |
|--------|-------------|
| [🔐 Login](LOGIN.md) | User authentication for Customers & Suppliers |
| [💊 Prescriptor](PRESCRIPTOR.md) | Patient prescription upload & medicine ordering |
| [🏪 Medical Store](MEDICAL_STORE.md) | Store order management & delivery flow |

## 🚀 How It Works

1. **Customer or Supplier** opens the app and signs up / logs in.
2. **Customer** uploads a prescription via the Prescriptor module.
3. **Medical Store** receives the order, checks availability, and dispatches via a rider.
4. **Supplier** manages medicine stock and sets validity.
